## Final checklist (Day 13)

- [ ] File opens as a standalone `.html` in Chrome
- [ ] File opens as a standalone `.html` in Firefox or Edge
- [ ] No external dependencies that require a network connection
- [ ] Flag is NOT in plain text anywhere in source
- [ ] Correct flag triggers success state
- [ ] Wrong flag triggers error state without revealing hints
- [ ] All three hints are accessible via the toggle
- [ ] Attempt counter or score display works (if implemented)
- [ ] Success badge/animation works
- [ ] No JavaScript console errors on load
- [ ] No JavaScript console errors on submission
- [ ] Text is readable at 100% zoom on a standard laptop screen
- [ ] Layout does not break at 1280px wide
