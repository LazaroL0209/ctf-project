Mirror Gate Lock

Difficulty: Beginner Points: 100 Category: Source Inspection
A decommissioned relay station has come back online — but its access panel is locked behind a calibration sequence no one remembers. Intercepted transmissions suggest the code was designed to be obvious to anyone paying attention. Once you crack the lock, the station begins broadcasting an encoded payload. Somewhere in the station's wiring, a verification fragment was hardcoded at installation.

Your mission: open the lock, recover the payload, find the hidden fragment, and reconstruct the full access token.

Need a nudge?
Hint 1: A palindrome reads the same forwards and backwards. You have three digits to work with — think about which arrangement is the smallest number that satisfies that rule.
Hint 2: Once the lock opens, two things become available. One is shown on the page. The other is hidden in the page's HTML structure — open your browser's Developer Tools (F12) and inspect the puzzle element.
Hint 3: The payload on screen is Base64-encoded. Open the browser console (F12 → Console tab) and run atob("the_payload_string") to decode it. Then find the data-key-frag attribute on the puzzle element and combine both pieces as FLAG{decoded_token_keyfrag}.
